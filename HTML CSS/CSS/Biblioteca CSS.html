Cores Complementares são as cores que mais tem Contrastecom a cor que voçê quer usar

Para descobrir qual é a cor complementar é so traçar uma reta no circulo cromatico ate a cor oposta a cor que voçê escolheu Ex: Para violeta a cor complementar é o Amarelo.

Cores Análogas

As cores Análogas são as cores de ambos os lados da cor Pricipal escolhida

Cores Análogas Relacionadas

Elas criam um degrader é nao tem contraste

Escolha uma cor, pegue a cor do lado qualquer lado, Pule uma cor e pegue a cor seguinte

Cores Intercaladas

escolha uma cor, pule a proxima cor, pegue a cor seguinte, pule a proxima de novo e pegue a seuinte de novo

Cores Triadicas

Escolha uma cor, Pule 3 cores e pegue a Proxima, Pule mais 3 Cores, e Pegue novamente a Proxima Cor

Cores em Quadrado

Escolha uma cor pule 2 cores e assim sucessivamente ate formar um Quadrado

Cores Tetradicas

Escolha uma cor, pegue a cor Complementar, Agora pegue qualquer outra cor e tambem pegue a sua cor Complementar

Cores Monocromia

É usada uma unica cor modificando a sua Saturação e o Brilho

Tamanho das fontes

Medidas Absolutas cm, mm, in, px, pt, pc

Medidas relativas em, ex, rem, vw, vh, %

Tamanho normal da fonte é 16px que é igual a 1em

A W3C recomenda sempre usar px mas de preferencia por usar (em)isso é bom porque voçê trabalha com % funciona melhor para tv celular tablet Alexa como parametro de tamanho das letras

Peso das Fontes

font-weight:lighter(mais magro), norma, bold(negrito), bolder(mais negrito que o negrito ou numerica(vai de 100 - 900)); É a peso da font

Fontes que não funcionam com font-weight -> Verdana, arial, times

font-style:italic ou normal; É italico ou normal

text-decoration:underline; Faz o efeito negrito italico sublinhado nem todas as fontes tem esse recurso

Configuraçoes de fontes muito Usadas, usadas até Demais

font-family:'work Sans' sans-serif;
         font-weight:bolder;
         font-size:3em(o 3 define o tamannho da letra);
         font-style:italic;

Propriedade shorthand
(Faz todas as configurações de uma letra tudo em uma unica linha)

ordem de criação do shorthand importante respeitar essa ordem

font-style(italico) ->font-weight(negrito) -> font-size(tamanho) -> font-family(tipo de fonte)
Ex: De como fica.
font: italic bolder 3em 'work Sans', sans-serif;
