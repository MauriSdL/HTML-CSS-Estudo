Listas

Tag li - - Defini um item da lista.

Toda lista começa com a Tag ul (lista não ordenada) ou Tag ol (lista ordenada).

Tag li dentro da Tag ul inicia a lista com um circulo Fechado Preenchido

Tag dl

Usada para criar um titulo comtendo sua descrição,
É usada em conjunto com as tags.
dt e dd.

A tag dt - - É o titulo.
A Tag dd - - É a descrição.(conteudo)

Titulo
Descição do titulo

Resultado

<dl> <dt>Titulo</dt> <dd>Descição do titulo</dd> </dl>
