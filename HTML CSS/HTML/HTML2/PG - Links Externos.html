Links Externos

Tag a

Cria um link externo clicavel tipo esse Acessar a Pagina referente ao Assunto, Clicando no link vai abrir a pagina chamada, porém em uma nova aba, Mantendo a pagina atual ainda em aberto..

OBS: Para chamar uma pagina CSS o type fica assim type=text/css

Codigo fica Assim: . . . . a href="link" target="_blank" rel="external"

Tag usada para definir uma associação entre um documento atual e um Externo.

O uso mais comum é para associar um documento CSS.


target="_blank" faz abrir o link em uma Aba nova, Mantendo sua pagina ainda em aberto

rel="external", Indica ao google que a pagina a ser aberta se refere a um link externo melhorando a busca no google

rel="nofollow"Diz para o mecanismo de busca(google) Para não dar o Aval, que perpetue a busca, Nesse caso o mecanismo de busca (Google), Não faz a indexação.
